# Project Rules Template

This document points to the canonical Deja Vu project rules template.

Use the copyable template in:

- [docs/templates/AGENTS.template.md](./templates/AGENTS.template.md)

Use the matching Markdown memory templates in:

- [docs/templates/memory](./templates/memory)

## Adoption notes

1. Replace `project:<project-id>` with a stable project scope id.
2. Keep the memory root at `memory/` unless the host has a stronger convention.
3. Preserve the recall, writeback, and compaction rules from the template.
4. If you later add the optional TypeScript engine, do not let it replace the Markdown files as the canonical source of truth.
