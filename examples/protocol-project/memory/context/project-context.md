# Example Protocol Project Context

## Product definition

This example demonstrates Deja Vu as a protocol-first memory system for agents working in one repository.

## Stable constraints

- no npm package is required for adoption
- the optional engine must not replace Markdown memory as the primary source of truth

## Boundaries

- in scope: rules, workflow, and Markdown memory conventions
- out of scope: autonomous memory daemons and global memory
