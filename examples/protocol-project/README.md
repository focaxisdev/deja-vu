# Protocol-First Example

This example shows how to adopt Deja Vu without npm or the optional TypeScript engine.

## Files

- `AGENTS.md`
- `memory/index.md`
- `memory/summary.md`
- `memory/context/project-context.md`
- `memory/decisions/`
- `memory/open-loops/`

## Flow

1. Add the rules file to the project.
2. Add the memory directory.
3. Use `memory/index.md` and `memory/summary.md` as the first recall surface.
4. Update detailed memory files only when new durable information appears.
